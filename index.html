Features

Everything in one CLI

Three commands cover the full SOC2 compliance lifecycle.

📋
cloudcompliance report
Reads terraform.tfstate, maps resources to SOC2 controls, generates JSON + Markdown evidence report.
v1.0.0
🔍
cloudcompliance drift
Compares live AWS state against Terraform state. Detects deleted or unmanaged resources. Fires SNS alert automatically.
v1.2.0
📈
cloudcompliance history
Every report run is saved to SQLite with timestamp. View score trend over time — exactly what SOC2 Type II auditors need.
v1.3.0 NEW
📤
cloudcompliance history --export
Exports full compliance timeline as JSON. Auditor-ready evidence file showing your security posture over time.

Quick start

Deploy in 3 commands

Requirements: Terraform, Docker

quick start
# 1. Start LocalStack (free local AWS)
$ docker run --rm -d -p 4566:4566 localstack/localstack:3.4.0

# 2. Deploy all 10 SOC2 controls
$ make deploy
Apply complete! Resources: 46 added, 0 changed, 0 destroyed.

# 3. Generate compliance evidence
$ make report

Evidence saved → compliance/compliance_report.json
Markdown report → compliance/compliance_report.md
